                       🩺 คู่มือผู้ป่วย (Patient Guide)
                          Hanna ผู้ช่วยดูแลสุขภาพ

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❤️ HANNA คืออะไร?

Hanna คือผู้ช่วยดูแลสุขภาพ AI ที่โรงพยาบาลของคุณใช้เพื่อช่วยติดตาม
สุขภาพของคุณทุกวัน

✅ Hanna จะ:
    • ส่งข้อความถามสุขภาพทุกวัน
    • เตือนเรื่องการกินยา
    • รับฟังอาการของคุณ
    • แจ้งพยาบาลถ้ามีอะไรผิดปกติ

❌ Hanna จะไม่:
    • ให้คำแนะนำทางการแพทย์
    • สั่งยา
    • วินิจฉัยโรค

⚠️ สำคัญ: Hanna ไม่ใช่แพทย์หรือพยาบาล 
   ถ้ามีเหตุฉุกเฉิน กรุณาโทร 1669 ทันที

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📱 วิธีใช้ HANNA

1️⃣ เพิ่มเพื่อนใน LINE

    1. เปิดแอป LINE
    2. ค้นหา @hanna หรือสแกน QR Code (ที่โรงพยาบาลให้)
    3. กด "เพิ่มเพื่อน"
    4. Hanna จะทักทายคุณ

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

2️⃣ ตอบคำถามทุกวัน

Hanna จะส่งคำถามพร้อมปุ่มให้กด:

    ขั้นตอนที่ 1: "วันนี้เป็นยังไงบ้าง?"
    ─────────────────────────────────────────────────────────────────────
    กดปุ่ม:  [ ดี 😊 ]  [ ปกติ 😐 ]  [ ไม่ดี 😟 ]

    ขั้นตอนที่ 2: "กินยาครบไหม?"
    ─────────────────────────────────────────────────────────────────────
    กดปุ่ม:  [ ได้ ครบ ✅ ]  [ บางส่วน ]  [ ไม่ได้เลย ]

    ขั้นตอนที่ 3: "มีอาการผิดปกติไหม?"
    ─────────────────────────────────────────────────────────────────────
    กดปุ่ม:  [ ไม่ มีแต่เรื่องดี 👍 ]  [ มีอาการบ้าง 😟 ]

💡 ง่ายมาก! แค่กดปุ่ม ไม่ต้องพิมพ์

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🏆 รางวัลความต่อเนื่อง

ถ้าคุณตอบคำถามทุกวัน Hanna จะฉลองให้:

    🔥 7 วัน:   "7 วันติดต่อกัน! สุดยอด!"
    🏆 14 วัน:  "2 อาทิตย์แล้ว! เก่งมาก!"
    ⭐ 30 วัน:  "1 เดือนเต็ม! ยอดเยี่ยม!"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

3️⃣ แจ้งอาการ (ถ้ามี)

ถ้าคุณกด "มีอาการบ้าง" Hanna จะแสดงปุ่มอาการให้เลือก:

    [ มีไข้ 🌡️ ]  [ ปวดศีรษะ ]  [ คลื่นไส้ ]
    [ หอบเหนื่อย ]  [ ปวด/บวม ]  [ เหนื่อยล้า ]

Hanna จะบันทึกและแจ้งพยาบาลให้คุณ

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

    ปุ่ม                    ทำอะไร

    ℹ️ ช่วยเหลือ            ดูคำสั่งที่ใช้ได้

🚨 เหตุฉุกเฉิน

⚠️ ถ้าคุณมีอาการเหล่านี้ กรุณาโทร 1669 ทันที:

    🚨 เจ็บหน้าอก
    🚨 หายใจไม่ออก
    🚨 เป็นลม
    🚨 หมดสติ
    🚨 อาการชัก

ถ้าคุณพิมพ์คำเหล่านี้ให้ Hanna จะแนะนำให้คุณโทร 1669 ด้วย

❓ คำถามที่พบบ่อย

❓ Hanna จะส่งข้อความบ่อยแค่ไหน?
   ➜ ประมาณวันละ 1-2 ครั้ง (เช้า และ/หรือ เย็น)

❓ ถ้าไม่ตอบจะเป็นอะไรไหม?
   ➜ ไม่เป็นไร แต่ถ้าไม่ตอบหลายวัน พยาบาลอาจโทรมาถามว่าคุณสบายดีไหม

❓ ข้อมูลของฉันปลอดภัยไหม?
   ➜ ปลอดภัย ข้อมูลของคุณถูกเก็บอย่างปลอดภัยและใช้เฉพาะทีมดูแลสุขภาพ

❓ ต้องการหยุดใช้ Hanna ทำยังไง?
   ➜ พิมพ์ "หยุด" หรือติดต่อโรงพยาบาลของคุณ

❓ Hanna ให้คำแนะนำทางการแพทย์ได้ไหม?
   ➜ ไม่ได้ สำหรับคำแนะนำทางการแพทย์ กรุณาติดต่อพยาบาลหรือแพทย์
